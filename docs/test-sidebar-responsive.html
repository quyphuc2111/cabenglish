Test Client Sidebar Responsive

Hướng dẫn test:

  1. Mở trang này trên desktop (width ≥ 1024px) - Sidebar sẽ slide vào từ trái
  2. Thu nhỏ width xuống dưới 1024px - Sidebar sẽ slide ra ngoài trái
  3. Khi sidebar ẩn, nút menu mobile sẽ xuất hiện ở góc trái trên
  4. Main content sẽ tự động điều chỉnh margin-left
  5. Lưu ý: Container có overflow-hidden để ẩn sidebar khi slide ra ngoài

Breakpoints:

  • lg: ≥ 1024px - Sidebar: translate-x-0 (hiển thị)
  • < lg: < 1024px - Sidebar: -translate-x-full (ẩn), mobile menu hiển thị
  • Container: overflow-hidden để ẩn sidebar khi slide ra ngoài
